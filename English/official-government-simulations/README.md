# Official government simulations — A2 Reading

This directory consolidates vocabulary, grammar structures, reading patterns, and exam traps observed in official/real A2 reading simulations for the Dutch Inburgering exam.

The focus is practical reading: notices, emails, tables, schedules, services, work, school, health, transport, events, shopping, and everyday rules.

## How to study

1. Read one topic at a time.
2. Mark the words you still cannot recognize quickly.
3. Study the grammar patterns and exam traps.
4. Complete the question bank without checking the answer key.
5. Review the small but dangerous words: `vanaf`, `voor`, `uiterlijk`, `tot en met`, `dan`, `daarna`, `als`, `omdat`, `hoeft niet`, `mag niet`.

## Mental checklist for A2 reading texts

| English | Dutch |
|---|---|
| Who? | Wie? |
| What? | Wat? |
| Where? | Waar? |
| When? | Wanneer? |
| What time? | Hoe laat? |
| From when? | Vanaf wanneer? |
| Until when? | Tot wanneer? |
| Before when? | Voor wanneer? |
| Why? | Waarom? |
| How much does it cost? | Hoeveel kost het? |
| Who has to do it? | Wie moet het doen? |
| What is not necessary? | Wat hoeft niet? |
| What is not allowed? | Wat mag niet? |
| What changed? | Wat is veranderd? |
| What stayed the same? | Wat blijft hetzelfde? |

## Included topics

- Work, shops, rosters, colleagues, holidays, and internal rules.
- School, courses, tests, certificates, presentations, and study methods.
- Health: dentist, GP, first aid, sickness reporting, and occupational doctor.
- Transport: buses, temporary routes, schedules, and free public transport.
- Services: hairdresser, furniture delivery, second-hand shop, and forgotten objects.
- Events: parties, weddings, sports day, neighborhood meeting, canoeing, and teacher farewell.

## Note

The examples came from official/real A2 practice simulations. This directory does not reproduce the full original texts. It transforms the material into vocabulary, patterns, and exam-style questions for A2 reading practice.
