# Gramática - Pontos Importantes A2

## Verbo na segunda posição

No holandês, em frases principais, o verbo costuma ficar na segunda posição.

- Ik werk vandaag thuis.
- Morgen werk ik thuis.

## Perguntas simples

- Waar woon je?
- Hoe laat begin je?
- Werk je morgen?

## Negação

- niet = usado para negar verbos, adjetivos e ideias gerais
- geen = usado com substantivos sem artigo indefinido

Exemplos:

- Ik werk niet vandaag.
- Ik heb geen auto.

## Passado comum na fala

- ik werkte
- hij woonde
- wij maakten

Não precisa dominar tudo de uma vez. Para A2, reconhecer e usar algumas formas frequentes já ajuda bastante.

## Verbos modais

- kunnen = poder
- moeten = precisar / dever
- willen = querer
- mogen = ter permissão

Exemplos:

- Ik kan morgen komen.
- Ik moet naar de huisarts.
- Wij willen Nederlands leren.

## Ordem com infinitivo

Quando há verbo modal, o segundo verbo vai para o fim.

- Ik wil Nederlands leren.
- Zij moet vandaag werken.

## Dica prática

Monte frases curtas com a mesma estrutura várias vezes. Repetição é mais útil do que decorar regra isolada.
