## Como usar no estudo

### Prioridade máxima para leitura A2

Use primeiro os PDFs `Oefentoets 1` até `Oefentoets 5`. Eles são simulados de leitura A2 com textos, perguntas de múltipla escolha e limite de tempo.

Sugestão:

1. faça uma prova inteira com cronômetro de 65 minutos;
2. corrija;
3. transforme os erros em vocabulário;
4. refaça os textos errados depois de alguns dias.

### Aquecimento

Use `Emailing 195-14-leesteksten-lezen-a1.pdf` quando estiver cansado ou quiser leitura mais fácil.

### Vocabulário

Use `Emailing e-book-lesson-1-20-1000DutchWords.pdf` como lista de palavras frequentes.

### Referência grande

Use `Emailing Inburgeringsexamen A2 (Ad Appel) (z-lib.org).pdf` como livro de referência, não como algo para estudar inteiro de uma vez.

## Observação prática

Para o seu caso, a ordem mais eficiente é:

1. `Oefentoets 1–5`
2. textos oficiais/simulados do governo já incorporados no repo
3. sessões de trem
4. micro-estudos no celular
5. vocabulário complementar
6. livro completo A2 como consulta
