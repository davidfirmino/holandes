# Seção de trem

Esta seção reúne materiais curtos para estudar **em trânsito**, principalmente no trem a caminho do trabalho.

A ideia é ter arquivos datados, fáceis de abrir no celular, com textos de leitura A2, perguntas, vocabulário e gabarito provável.

## Como usar no trem

1. Leia um texto sem traduzir palavra por palavra.
2. Responda às perguntas primeiro sem olhar o gabarito.
3. Marque as palavras que aparecem sempre: `ongeveer`, `volgens`, `daarom`, `voordat`, `terwijl`, `sinds`, `toch`, `namelijk`.
4. Depois confira o gabarito provável e releia o trecho que prova a resposta.
