# Como usar este recurso externo no seu plano

Este material é útil, mas precisa entrar no plano sem bagunçar sua prioridade atual.

Sua prioridade agora é clara:

> **Passar bem na leitura A2 e ganhar independência do visto de trabalho.**

Então este diretório deve ser usado como **apoio**, não como substituto do treino estilo prova.

---

## Regra principal

Não transforme isso em mais uma montanha de PDF para carregar culpa.

Use assim:

- se você tem **15 minutos no celular**: use `11-micro-estudos-celular/`;
- se você está **no trem**: use `10-sessao-trem/`;
- se você quer **simulado mais próximo da prova**: use `08-simulados-oficiais-governo/`;
- se você quer **revisar vocabulário/gramática/visão geral**: use este diretório externo.

---

## Ordem sugerida

### 1. Lezen — agora

Use a dica do README original:

> leia as perguntas primeiro, depois procure no texto.

Workflow:

1. Leia as perguntas.
2. Sublinhe mentalmente as palavras-chave.
3. Leia o texto procurando essas palavras ou sinônimos.
4. Elimine alternativas parcialmente corretas.
5. Responda.
6. Só depois traduza palavras desconhecidas.

### 2. Vocabulário — junto com leitura

Use `vocabulaire.pdf` quando perceber que está errando por falta de vocabulário básico.

Não leia o PDF inteiro passivamente. Melhor usar assim:

1. escolha uma página;
2. pegue 10 palavras;
3. faça frases A2 simples;
4. tente reconhecer essas palavras nos exercícios.

### 3. Gramática — quando um padrão travar

Use `basicgrammer_syll.pdf` apenas quando aparecer uma dúvida real, por exemplo:

- inversão depois de tempo/lugar;
- ordem do verbo;
- separable verbs;
- `omdat` jogando o verbo para o fim;
- passado/perfectum;
- pronomes e possessivos.

### 4. Schrijven — depois da leitura estar sob controle

Quando for treinar escrita, use as dicas do README original:

- frases simples;
- saudações prontas;
- encerramentos prontos;
- estrutura clara.

Exemplos úteis:

- `Bedankt voor je hulp.`
- `Ik wil graag weten...`
- `Het spijt me hiervoor.`
- `Met vriendelijke groet,`

### 5. Spreken — fase separada

Os PDFs de speaking são úteis, mas não agora como prioridade máxima. Use quando quiser começar blocos de fala de 10 minutos.

### 6. KNM — fase futura

KNM parece mais fácil para quem já vive na Holanda há anos, mas ainda precisa treino de perguntas. Guarde o PDF de 1000 questões para uma fase específica.

## Plano semanal simples

### Segunda a sexta

- 30–50 min de leitura pela manhã, como você já faz.
- 1 micro-estudo no celular quando der.

### No trem

- 1 sessão de `10-sessao-trem/`.

### Fim de semana

- 10–20 min leve.
- Revisar vocabulário, sem pressão.

---

## Frase anti-culpa

Um bloco de 15 minutos conta.

Não é pouco. É consistência.

Para A2, contato frequente com texto simples vale muito mais do que estudar três horas uma vez e depois sumir por quatro dias.
