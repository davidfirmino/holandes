# Centralized vocabulary with A2 sentences — Lessons 1 to 26

This section centralizes the essential vocabulary from lessons 1 to 26 in a more useful format for reading and memorization: **Dutch word + English translation + A2 example sentence + sentence translation**.

The goal is to avoid memorizing isolated words. In the A2 exam, you need to recognize vocabulary inside simple sentences, notices, emails, tables, and instructions.

## How to study

1. Read the Dutch word.
2. Read the A2 sentence out loud.
3. Cover the translation and try to understand from context.
4. Write a similar sentence about your own life.
5. Review the small high-impact words first: `vanaf`, `voor`, `tot`, `omdat`, `als`, `dan`, `moet`, `mag`, `kan`, `hoeft niet`.
