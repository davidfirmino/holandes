# Vocabulário de Alta Frequência - A2

Este arquivo resume as palavras que aparecem com muita frequência no dia a dia e no exame.

## Perguntas

- wat = o que
- waar = onde
- wie = quem
- wanneer = quando
- waarom = por que
- hoe = como
- welke = qual / quais
- hoeveel = quanto(s)

## Verbos centrais

- zijn = ser / estar
- hebben = ter / possuir
- gaan = ir / ir fazer
- komen = vir
- doen = fazer
- maken = fazer / criar
- wonen = morar
- werken = trabalhar
- leren = aprender
- lezen = ler
- schrijven = escrever
- spreken = falar
- luisteren = ouvir
- eten = comer
- drinken = beber

## Verbos modais

- kunnen = poder / conseguir
- moeten = ter que
- willen = querer
- mogen = ter permissão

## Conectores

- en = e
- maar = mas
- of = ou
- want = porque, ordem normal
- omdat = porque, verbo no final
- dus = então / por isso
- daarom = por isso
- als = se / quando
- dan = então / depois / do que

## Tempo

- nu = agora
- vandaag = hoje
- morgen = amanhã
- gisteren = ontem
- straks = daqui a pouco
- vroeger = antigamente / antes
- tegenwoordig = hoje em dia
- geleden = há / atrás

## Frequência

- altijd = sempre
- vaak = com frequência
- soms = às vezes
- nooit = nunca

## Palavras pequenas, mas importantes

- nog = ainda / mais
- al = já
- weer = de novo / tempo
- mee = junto
- terug = de volta
- weg = embora / sumido
- bijna = quase
- pas = só / apenas

## Dica

Não estude essas palavras isoladas por muito tempo. Monte frases curtas com elas:

- Vandaag werk ik niet.
- Ik leer Nederlands omdat ik in Nederland woon.
- Neem je je jas mee?
