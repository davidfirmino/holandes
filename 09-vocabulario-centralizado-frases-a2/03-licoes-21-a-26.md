# Lições 21 a 26 — Vocabulário centralizado com frases A2

Formato: **palavra em holandês + tradução + frase A2 + tradução da frase**.

> Observação: este bloco funciona como consolidação prática para o final do A2. Ele usa vocabulário típico de comunicação, formulários, serviços, dinheiro, saúde prática, trabalho e exame.

## Les 21 — Communicatie / Comunicação

|---|---|---|---|

| de e-mail | o e-mail | Ik schrijf een e-mail. | Eu escrevo um e-mail. |

| bellen | ligar | Ik bel mijn collega. | Eu ligo para meu colega. |
| opnemen | atender | Neem de telefoon op, alstublieft. | Atenda o telefone, por favor. |
| terugbellen | ligar de volta | Kunt u mij terugbellen? | Pode me ligar de volta? |
| sturen | enviar | Ik stuur de foto’s vandaag. | Eu envio as fotos hoje. |
| ontvangen | receber | Ik ontvang een brief. | Eu recebo uma carta. |
| antwoorden | responder | Antwoord vóór vrijdag. | Responda antes de sexta. |
| vragen | perguntar | Ik wil iets vragen. | Quero perguntar algo. |
| vertellen | contar / dizer | Vertel mij wat er is gebeurd. | Conte-me o que aconteceu. |
| bespreken | discutir / conversar sobre | We bespreken het rooster. | Nós discutimos a escala. |
| uitleggen | explicar | De docent legt de regel uit. | O professor explica a regra. |
| begrijpen | entender | Ik begrijp de tekst. | Eu entendo o texto. |
