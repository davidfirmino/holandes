# Consolidado — Leitura A2

Esta pasta contém os materiais centrais para revisão do exame **Lezen A2 / Inburgering**.

## Arquivo principal

- [GUIA_COMPLETO_LEITURA_A2.md](./GUIA_COMPLETO_LEITURA_A2.md)

Este guia reúne:

- estratégia de prova para leitura A2;
- palavras que mais criam pegadinhas;
- gramática essencial para leitura;
- vocabulário consolidado por tema;
- padrões de frases comuns em cartas, avisos, e-mails e anúncios;
- checklist antes de responder;
- plano de revisão;
- mini teste de sobrevivência A2.

## Como usar

Use este material como ponto de partida antes de fazer simulados.

Ordem sugerida:

1. Leia a seção de estratégia.
2. Revise as palavras de pegadinha.
3. Revise a gramática essencial.
4. Escolha um tema de vocabulário.
5. Faça um simulado em `07-exercicios`.
6. Volte para este guia para revisar os erros.

## Foco

O foco desta pasta é **leitura**, não conversação perfeita.

Para o exame A2, o mais importante é reconhecer rapidamente:

- datas;
- horários;
- preços;
- lugares;
- obrigações;
- proibições;
- permissões;
- exceções;
- motivos;
- conselhos;
- ações que você precisa fazer.
