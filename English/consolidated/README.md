# Consolidated — A2 Reading

This folder contains the central materials for reviewing **Lezen A2 / Inburgering**.

## Main file

- [COMPLETE_A2_READING_GUIDE.md](./COMPLETE_A2_READING_GUIDE.md)

This guide brings together:

- A2 reading exam strategy;
- common trap words;
- essential reading grammar;
- consolidated vocabulary by theme;
- common sentence patterns in letters, notices, e-mails, and ads;
- checklist before answering;
- revision plan;
- A2 survival mini-test.

## How to use it

Use this material as a starting point before doing simulations.

Suggested order:

1. Read the strategy section.
2. Review the trap words.
3. Review the essential grammar.
4. Pick one vocabulary theme.
5. Do a simulation in `English/exercises` or `07-exercicios`.
6. Return to this guide to review mistakes.

## Focus

The focus of this folder is **reading**, not perfect conversation.

For the A2 exam, the most important thing is to recognize quickly:

- dates;
- times;
- prices;
- places;
- obligations;
- prohibitions;
- permissions;
- exceptions;
- reasons;
- advice;
- actions you need to take.
