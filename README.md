## Como usar

1. Estude um tema de vocabulário por vez.
2. Revise um ponto de gramática junto com frases reais.
3. Escreva pequenas respostas usando os modelos.
4. Marque o que já domina na lista de revisão.

## Arquivo central de revisão

O arquivo [DUTCH_A2_MASTER_REVIEW.md](/home/davidfr/holandes/DUTCH_A2_MASTER_REVIEW.md:1) agora entra como seu material de revisão ampla. Ele é útil para:

- revisão rápida antes de estudar
- consolidar vocabulário de alta frequência
- revisar contrastes importantes como `want` vs `omdat`
- treinar leitura para o estilo do exame

## Foco do nível A2

No A2, a meta não é falar perfeito. A meta é conseguir:

- se apresentar
- falar da rotina
- marcar horário
- entender instruções simples
- escrever mensagens curtas
- conversar sobre trabalho, casa, saúde e compras
