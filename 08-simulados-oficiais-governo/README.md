# Simulados oficiais do governo - Leitura A2

Este diretório consolida vocabulário, estruturas e padrões de pergunta observados nos simulados oficiais/reais de leitura A2 do governo para o exame de Inburgering.

O foco é leitura prática: avisos, e-mails, tabelas, horários, serviços, trabalho, escola, saúde, transporte, eventos, compras e regras do dia a dia.

## Como estudar

1. Leia um tema por vez.
2. Marque as palavras que ainda não reconhece rapidamente.
3. Depois leia os padrões de gramática e pegadinhas.
4. Faça o banco de questões sem olhar o gabarito.
5. Revise as palavras pequenas: `vanaf`, `voor`, `uiterlijk`, `tot en met`, `dan`, `daarna`, `als`, `omdat`, `hoeft niet`, `mag niet`.

## Checklist mental para resolver textos A2

| Português | Holandês |
|---|---|
| Quem? | Wie? |
| O quê? | Wat? |
| Onde? | Waar? |
| Quando? | Wanneer? |
| Que horas? | Hoe laat? |
| A partir de quando? | Vanaf wanneer? |
| Até quando? | Tot wanneer? |
| Antes de quando? | Voor wanneer? |
| Por quê? | Waarom? |
| Quanto custa? | Hoeveel kost het? |
| Quem precisa fazer? | Wie moet het doen? |
| O que não precisa? | Wat hoeft niet? |
| O que não pode? | Wat mag niet? |
| O que mudou? | Wat is veranderd? |
| O que ficou igual? | Wat blijft hetzelfde? |

## Temas incluídos

- Trabalho, loja, escala, colegas, férias e regras internas.
- Escola, cursos, provas, certificados, apresentações e métodos de estudo.
- Saúde: dentista, huisarts, EHBO, ziekmelding e bedrijfsarts.
- Transporte: ônibus, rotas temporárias, horários e gratuidade.
- Serviços: cabeleireiro, entrega de móveis, kringloopcentrum e objetos perdidos.
- Eventos: festa, casamento, sportdag, buurtbijeenkomst, kanovaren e despedida de professor.

## Observação

Os exemplos vieram de simulados oficiais/reais. Este diretório não tenta reproduzir os textos integralmente; ele transforma o material em vocabulário, padrões e perguntas de treino para leitura A2.
