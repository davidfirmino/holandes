# Modelos de Texto

## 1. Mensagem simples

Hallo,

Ik kan morgen niet komen. Ik ben ziek.
Kunnen we een nieuwe afspraak maken?

Met vriendelijke groet,  
David

## 2. E-mail para trabalho

Beste [naam],

Ik ben morgen iets later op het werk.
Mijn trein heeft vertraging.
Ik verwacht om 09:15 uur aan te komen.

Met vriendelijke groet,  
[jouw naam]

## 3. Texto sobre você

Ik heet [naam].
Ik kom uit [land].
Ik woon in [stad].
Ik werk als [beroep] of ik zoek werk.
In mijn vrije tijd lees ik, sport ik of luister ik naar muziek.

## Como praticar

- troque as informações pelos seus dados
- reescreva o mesmo modelo sem olhar
- leia o texto em voz alta
