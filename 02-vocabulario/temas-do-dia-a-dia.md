# Vocabulário - Temas do Dia a Dia

## Apresentação

- ik heet... = eu me chamo...
- ik kom uit Brazilië = eu venho do Brasil
- ik woon in... = eu moro em...
- ik ben getrouwd = eu sou casado
- ik ben alleenstaand = eu sou solteiro

## Casa

- het huis = a casa
- de kamer = o quarto / cômodo
- de keuken = a cozinha
- de badkamer = o banheiro
- de huur = o aluguel
- de sleutel = a chave

## Trabalho

- het werk = o trabalho
- de collega = o colega
- de baas = o chefe
- werken = trabalhar
- zoeken = procurar
- fulltime = tempo integral
- parttime = meio período

## Saúde

- de huisarts = o clínico geral / médico de família
- de afspraak = o agendamento
- ziek = doente
- pijn = dor
- de apotheek = a farmácia

## Compras

- de winkel = a loja
- goedkoop = barato
- duur = caro
- betalen = pagar
- contant = em dinheiro
- pinnen = pagar com cartão

## Transporte

- de trein = o trem
- de bus = o ônibus
- de halte = a parada
- te laat = atrasado
- op tijd = no horário

## Frases curtas

- Ik zoek werk.
- Ik heb een afspraak.
- Hoe laat vertrekt de trein?
- Waar is de supermarkt?
