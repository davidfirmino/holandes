# 3. Números de 21 a 99

Aqui está uma das maiores pegadinhas para brasileiros: em holandês, de 21 a 99, a unidade vem antes da dezena.

Em português:

> vinte e um

Em holandês:

> um e vinte

Ou seja:

**21 = eenentwintig**  
Literalmente: um-e-vinte.

## Estrutura

```text
unidade + en + dezena
```

Exemplos:

| Número | Holandês | Literal |
|---:|---|---|
| 21 | eenentwintig | um e vinte |
| 22 | tweeëntwintig | dois e vinte |
| 23 | drieëntwintig | três e vinte |
| 24 | vierentwintig | quatro e vinte |
| 25 | vijfentwintig | cinco e vinte |
| 26 | zesentwintig | seis e vinte |
| 27 | zevenentwintig | sete e vinte |
| 28 | achtentwintig | oito e vinte |
| 29 | negenentwintig | nove e vinte |

## Outros exemplos importantes

| Número | Holandês |
|---:|---|
| 31 | eenendertig |
| 32 | tweeëndertig |
| 35 | vijfendertig |
| 41 | eenenveertig |
| 48 | achtenveertig |
| 52 | tweeënvijftig |
| 67 | zevenenzestig |
| 74 | vierenzeventig |
| 86 | zesentachtig |
| 99 | negenennegentig |

## O trema em tweeën / drieën

Em alguns números aparece `ë`, como em:

- tweeëntwintig = 22
- drieëndertig = 33
- tweeënveertig = 42

Esse trema mostra que você deve pronunciar separado:

```text
twee-en-twintig
```

Não precisa se desesperar com isso no começo. Para leitura A2, o mais importante é reconhecer a estrutura.

# 10. Telefone

Números de telefone geralmente são lidos em grupos.

Exemplo fictício:

```text
06 - 12 34 56 78
```

Pode ser lido como:

```text
nul zes, twaalf, vierendertig, zesenvijftig, achtenzeventig
```

Para o A2, o mais importante é reconhecer:

| Holandês | Português |
|---|---|
| telefoonnummer | número de telefone |
| bel ons op | ligue para nós em |
| bereikbaar op | contactável em |
| tussen 8.00 en 12.00 uur | entre 8h e 12h |
| vanaf 7.00 uur | a partir das 7h |
| voor 9.30 uur | antes das 9h30 |

## Frases A2

| Nederlands | Português |
|---|---|
