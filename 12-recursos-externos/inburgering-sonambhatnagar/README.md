# Recurso adicional — sonambhatnagar/inburgering

**Fonte original:** `sonambhatnagar/inburgering`  
**Tema:** recursos gerais para o exame Inburgering  
**Uso no nosso repo:** material complementar para consulta, especialmente leitura, escrita, fala, listening e KNM.

> Observação importante: este diretório funciona como uma incorporação organizada do repositório externo dentro do nosso material de estudos. Mantive o README original copiado em `ORIGINAL_README.md` e criei um manifesto com todos os arquivos e links diretos. Os PDFs continuam referenciados a partir da fonte original, para preservar autoria, origem e evitar duplicação binária pesada no nosso repo.

---

## Por que esse recurso é útil

Esse repo é interessante porque reúne materiais práticos de alguém que passou pelo processo do Inburgering e organizou recursos por parte do exame:

- **Lezen** — leitura;
- **Luisteren** — escuta;
- **Schrijven** — escrita;
- **Spreken** — fala;
- **KNM** — Knowledge of Dutch Society;
- **ONA** — Orientation to the Dutch Labour Market.

Para o nosso objetivo atual, que é priorizar **leitura A2**, a parte mais relevante é a estratégia mencionada no README original: ler as perguntas primeiro, depois procurar as respostas nos textos. Isso combina muito com os exercícios que estamos montando nas seções de trem e micro-estudos.

## Melhor forma de usar para você

Como seu foco agora é passar bem em **leitura A2**, eu usaria assim:

1. Continue usando as sessões de trem e micro-estudos como treino principal.
2. Use o material externo para complementar gramática, vocabulário e escrita.
3. Não troque o treino estilo prova por leitura passiva de PDF.
4. Para leitura, aplique sempre a técnica: **pergunta primeiro, texto depois**.
5. Para speaking/escrita, use os PDFs do repo externo depois que a leitura estiver mais estável.
